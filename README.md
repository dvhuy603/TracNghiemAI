# 🎯 HỆ THỐNG TRẮC NGHIỆM THÔNG MINH (TRÍ TUỆ NHÂN TẠO)

## 📌 Giới thiệu
Đây là chương trình trắc nghiệm được xây dựng bằng ngôn ngữ C++.  
Hệ thống cho phép người dùng làm bài trắc nghiệm, chấm điểm tự động và hiển thị kết quả sau khi hoàn thành.

---

## ⚙️ Chức năng chính
- Làm bài trắc nghiệm ngẫu nhiên  
- Làm bài theo độ khó (1–3)  
- Nhập và lưu đáp án người dùng  
- Chấm điểm tự động  
- Hiển thị kết quả đúng/sai  
- Đọc dữ liệu câu hỏi từ file JSON  

---

## 🧠 Công nghệ sử dụng
- Ngôn ngữ: C++  
- Thư viện: STL (vector, iostream, fstream...)

## 🚀 Cách chạy chương trình

Biên dịch:
g++ main.cpp docjson.cpp lambai.cpp danhgia.cpp -o tracnghiem  

Chạy:
./tracnghiem
